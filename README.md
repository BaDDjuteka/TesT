# 한국 주식 분석 및 자동 매매 신호

KRX 종목 코드나 업비트 KRW 코인 마켓을 입력하면 가격 차트, 이동평균선, RSI, MACD, 거래량 조건을 계산해 참고용 매매 신호를 보여주는 Streamlit 앱입니다.
옵션을 켜면 커뮤니티 분위기와 뉴스 분위기를 분리해서 비교합니다. 뉴스는 네이버, 네이트, 다음, 주요 금융 뉴스 묶음을 함께 봅니다.

## 실행

```powershell
pip install -r requirements.txt
streamlit run app.py
```

## 비공개 웹 실행

`APP_PASSWORD`를 설정하면 앱 접속 전에 비밀번호를 요구합니다.

```powershell
$env:APP_PASSWORD="원하는_비밀번호"
streamlit run app.py
```

클라우드 배포 방법은 `DEPLOYMENT.md`를 참고하세요.

## 주식 검색

주식은 종목 이름이나 6자리 코드를 입력해 검색할 수 있습니다.
띄어쓰기와 일부 약칭을 보정하고, 정확히 일치하지 않아도 비슷한 이름을 후보로 보여줍니다.

- `삼성전자`
- `삼전`
- `셀트리온`
- `하닉`
- `005930`

검색은 FinanceDataReader의 KRX 상장 목록과 네이버 금융 온라인 검색 결과를 사용합니다. 앱에 고정된 내장 종목 목록은 검색 결과로 사용하지 않습니다.

## 최근 본 항목

분석한 주식과 코인은 `view_history.json`에 저장됩니다. 사이드바의 `최근 본 항목`에서 다시 선택할 수 있습니다.

## 코인 코드 예시

- `KRW-BTC`: 비트코인
- `KRW-ETH`: 이더리움
- `KRW-XRP`: 리플
- `KRW-SOL`: 솔라나

코인은 업비트 공개 일봉 API를 사용합니다.

## 분위기 비교

- 주식 커뮤니티: 네이버 종목토론실, 토스 수집 상태
- 코인 커뮤니티: 코인톡 공개 글
- 뉴스: 네이버뉴스, 네이트뉴스, 다음뉴스, 주요 금융 뉴스 묶음

제목의 긍정/부정 키워드를 단순 집계합니다. 커뮤니티와 뉴스는 선동성, 반복 글, 늦은 반응이 섞일 수 있습니다.

## 단기 후보 스캔

사이드바의 `단기 후보 스캔` 버튼을 누르면 입력한 검색어를 바탕으로 온라인 후보군을 만든 뒤 기술적 신호 점수가 높은 후보를 보여줍니다.
이는 자동 매수 추천이 아니라 추가 확인용 관심 목록입니다.

- 주식: FinanceDataReader KRX 목록과 네이버 금융 온라인 검색 결과
- 코인: 업비트 KRW 마켓 목록

## 손실 최소화 계획

사이드바의 리스크 관리 값을 기준으로 각 종목의 손절 기준과 진입 규모를 계산합니다.

- 운용 자금
- 1회 허용 손실
- 종목당 최대 투입 비중
- ATR 기반 손절 배수

분석 결과에는 기준 손절가, 예상 최대손실, 계산 수량, 20일 변동성, 60일 고점 대비 낙폭, 위험 등급이 표시됩니다.

## 수익성 강화 지표

손절 기준을 바탕으로 목표가와 손익비를 함께 계산합니다.

- 1차 목표가: ATR 또는 20일 고점 기준
- 2차 목표가: ATR 2배 또는 60일 고점 기준
- 손익비: 목표 수익폭을 손절폭으로 나눈 값
- 20일/60일 고점 여지
- 5일 모멘텀
- 수익성 등급

단기 후보 스캔 표에도 수익성점수, 5일 수익률, 60일 고점 여지를 표시합니다.

## 신호 로직

- 추세: 종가, 20일 이동평균, 60일 이동평균
- 리스크: ATR20, 20일 변동성, 60일 고점 대비 낙폭, 20일선 이격
- 수익성: 목표가, 손익비, 5일 모멘텀, 20일/60일 고점 여지
- 단기 모멘텀: 5일선과 20일선의 골든크로스/데드크로스
- MACD: MACD와 시그널선 교차
- RSI: 과매도/과열 구간
- 거래량: 20일 평균 거래량 대비 급증과 20일 고점권 여부
- 커뮤니티/뉴스 참고: 주식과 코인별 공개 소스 분위기 비교

이 신호는 투자 조언이 아니라 분석 보조 도구입니다. 실제 매매에는 손절 기준, 포지션 크기, 뉴스/공시, 재무 지표, 시장 상황을 함께 반영해야 합니다.
